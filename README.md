# Florería Krystal — Sistema de Inventarios (Frontend)

App React para gestión básica de inventario: productos, movimientos, reportes y sprint backlog.

## Características
- Login simulado
- CRUD de productos (alta/editar/eliminar)
- Registro de movimientos (ingresos/salidas)
- Dashboard con resumen de stock
- Exportar reportes CSV
- Visualización del backlog y calendario de sprints

## Ejecutar localmente
1. `npm install`
2. `npm start` (http://localhost:5174/))

## Despliegue
- Recomendado: Vercel (conectar repo GitHub)
- Alternativa: GitHub Pages (`gh-pages`)
